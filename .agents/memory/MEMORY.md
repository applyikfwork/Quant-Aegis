- [Orval query-param collision](orval-query-param-collision.md) — Query params generate `<OperationId>QueryParams` in both api.ts and types/, causing TS2308; use path params or remove query params to avoid.
- [Bybit API volume field](bybit-api.md) — Use `turnover24h` (not `volume24h`) for USD-denominated volume; `volume24h` is in base asset units. Klines are reverse-chronological, must `.reverse()` before use. Interval map: 1m→"1", 5m→"5", 1h→"60", 4h→"240".
- [Supabase TCP blocked](supabase-setup.md) — Replit sandbox blocks direct TCP to PostgreSQL (port 5432). Must use Supabase JS client (REST/HTTP). Management API for CREATE TABLE needs a personal access token, not service_role JWT — schema must be applied manually via Supabase SQL Editor using supabase-setup.sql in project root.
